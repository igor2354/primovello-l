Как определить свой типовой размер:
Для курток, пальто, шуб основной параметр - обхват груди.
Для брюк, основной параметр - обхват талии и бедер.
Как определить свои параметры:
  • обхват груди
    Сантиметровая лента проходит по наиболее выступающим точкам груди, далее - подмышечными впадинами, на спине - немного выше.
  • обхват талии
    Измерятется строго по талии.
  • обхват бедер
    Сантиметровая лента проходит по наиболее выступающим точкам ягодиц.
